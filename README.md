# TodoApp

This app lets you:
1. Create tasks
2. Read tasks
3. Update tasks
4. Delete tasks


## Features
- Simple form input
- Simple list that display entered tasks
- Data stored in `localStorage`
- A button to delete all tasks
- A button to edit task
- A button to delete each task
- A button to toggle task as completed
